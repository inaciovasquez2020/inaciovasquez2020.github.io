# Research Portal Quickstart

This is the shortest path from clone to a first successful local verification pass.

## Requirements

- `git`
- `bash`
- `python3`
- `make`

## 1. Clone

```bash
git clone https://github.com/inaciovasquez2020/inaciovasquez2020.github.io.git
cd inaciovasquez2020.github.io
```

## 2. Check tools

```bash
python3 --version
git --version
make --version
```

## 3. Run the canonical repository check

```bash
make verify
```

## 4. Review the main public surfaces

- `README.md`
- `index.html`
- `publications.html`
- `contact.html`
- `docs/`
